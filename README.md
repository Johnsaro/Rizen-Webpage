# Rizen Showcase

A dedicated showcase website for the **Rizen** project, a Flutter-based Android application that gamifies real-life activities into RPG-style quests.

This repository contains the source code for the showcase site, designed to highlight features, progress, and the overall vision of Rizen.

## Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Linting:** [ESLint](https://eslint.org/)

## Getting Started

To run the showcase site locally, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## About Rizen
Rizen is a productivity-focused mobile application that transforms daily tasks and habits into an engaging, gamified RPG experience.
